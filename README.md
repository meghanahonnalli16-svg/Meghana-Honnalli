# 📦 Warehouse Inventory Robot (Simulation)

## 📌 Overview

This project simulates a warehouse inventory robot using Python. It demonstrates how inventory is managed using sorting, searching, and data analysis.

---

## 🚀 Features

* Inventory stored using Python lists
* Binary search for SKU lookup
* Sorting using bucket sort concept
* NumPy for stock analysis
* Detects slow-moving items

---

## 🛠️ Technologies Used

* Python
* NumPy

---

## ▶️ How to Run

```
pip install numpy
python warehouse_inventory_robot.py
```
